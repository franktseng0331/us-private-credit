# 美国私募信贷数据集说明文档

## 1. 数据概览

### 1.1 基本信息
- **数据来源**: SEC EDGAR数据库 - BDC（业务发展公司）10-Q/10-K申报文件
- **数据时间范围**: 2021-01-06 至 2026-02-27
- **总记录数**: 73,960条
- **覆盖BDC数量**: 50个（实际有数据39个）
- **借款人数量**: 8,570家企业
- **季度覆盖**: 21个季度
- **数据生成时间**: 2026-04-15

### 1.2 数据结构
每条记录代表一个BDC在某个季度对某家企业的单笔贷款持仓，包含以下信息：
- BDC基本信息（CIK、名称、股票代码）
- 申报文件信息（文件类型、提交日期、报告期）
- 借款人信息（企业名称、行业分类）
- 贷款条款（投资类型、优先级、利率、到期日）
- 财务数据（持仓规模、成本基础、公允价值）
- 元数据（数据来源格式、解析时间）

## 8. 数据使用建议

### 8.1 适用场景
✅ **适合**:
- 私募信贷市场规模和趋势分析
- BDC投资组合构成研究
- 借款人行业分布分析
- 贷款条款（利率、期限、优先级）统计
- 时间序列分析（季度变化）

⚠️ **需谨慎**:
- 绝对金额分析（存在单位解析问题）
- 行业分类分析（10%+数据行业字段质量差）
- 跨BDC对比（不同BDC表格格式差异大）

### 8.2 数据清洗建议
1. **标准化投资类型**: 合并同义词（如"First Lien Term Loan"与"First lien senior secured loan"）
2. **修正行业字段**: 清除被错误填充为投资类型的记录，重新分类
3. **验证金额单位**: 检查异常大额记录，确认原始文件中的单位标注
4. **补充缺失数据**: 对于零覆盖的11个BDCs，检查CIK是否正确，或尝试手动解析

### 8.3 后续扩展方向
- 增加更多BDCs（目前仅覆盖Top 50）
- 延长时间跨度（向前追溯至2015年）
- 补充8-K重大事件申报数据
- 整合FRED宏观经济数据
- 添加借款人信用评级信息

## 9. 技术实现说明

### 9.1 数据采集
- **工具**: edgartools库（Python）
- **数据源**: SEC EDGAR API
- **速率限制**: 10请求/秒
- **文件格式**: iXBRL（2022年后）+ HTML（2021-2022年）

### 9.2 解析方法
- **XBRL解析器**: 用于2022年后的结构化数据
- **HTML解析器**: 用于2021-2022年的表格数据
- **表格识别**: 基于关键词匹配（"Schedule of Investments"）

### 9.3 已知限制
- 部分BDC使用非标准表格格式，解析失败
- OXLC（Oxford Lane Capital）作为CLO基金不提交10-Q文件
- 不同BDC的字段命名不一致，需要映射规则
